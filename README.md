## 安装
```
  git clone https://github.com/Jon-Millent/faker.git
  cd faker
  npm install
  npm run dev
```
## 模块
* 首页
  * 微信
    * 生成微信首页
    * 生成微信聊天
    * 生成微信红包
    * 生成微信转账
    * 生成微信零钱
  * 支付宝
    * 生成支付宝账单
    * 生成支付宝余额
    * 生成余额宝
    * 生成花呗
* 我的（没对接后台）
  * 登录
  * 注册
* 设置
  * 关于
